# Q-Learning CRUD API

A simple FastAPI application for managing Q-values in a reinforcement learning environment (Q-learning).

## Features
- Create, Read, Update, Delete Q-values in SQLite
- Train a Q-learning agent via API
- Easily extensible for more complex RL tasks

## Prerequisites
- Python 3.9+ installed
- pip installed
- Internet connection to install dependencies
